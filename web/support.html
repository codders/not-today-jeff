How do I install the browser extension?

Firefox

You should be able to find the browser extension in the Firefox Addons Store. Click the Add to Firefox button to add it to your browser

Chrome

You should be able to find the browser extension in the chrome web store. Click the Add to Firefox button to add it to your browser

I have a problem with or question about the browser extension. Who do I contact?

You can contact open an issue on the Github Issues page for the project.

How can I be sure that there isn't some bad code in the browser extension?

You can build the extension yourself from the source code on Github. The builds we provide are just for your convenience.
